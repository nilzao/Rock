Rock Framework

Classes de uso comum para aplicações php.
  • Autoload -> Spl Autoload de classes PHP FIG 2.0
  • Front -> chamada das rotas de entrada
  • IController -> interface que os controllers das aplicações precisam implementar
  • Input -> tratamento de $_REQUEST, $_POST, $_GET e Shell args
  • Session -> manipuladora de $_SESSION
  • ViewLoader -> chama camada view detectando navegador/shell ou formato forçado na rota de entrada
